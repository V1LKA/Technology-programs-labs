# Лабораторная работа №13 — Коллекции и структуры данных

## Цель работы
Изучить использование коллекций JavaScript (Array, Map) для хранения, обработки, поиска, удаления и сортировки данных.

## Описание проекта
Веб-приложение реализует список студентов и позволяет:
- добавлять новые элементы,
- удалять элементы,
- искать по имени,
- сортировать коллекцию по алфавиту.
