# Лабораторная работа №11 — Введение во фреймворки (React)

## ✅ Цель работы
Освоить компонентный подход в React, работу со состоянием, обработкой событий и передачей данных между компонентами.

## ✅ Описание приложения
Простое SPA-приложение, позволяющее:
- выводить список студентов
- добавлять нового студента через форму
- удалять студента
